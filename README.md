# 🐦 Twitter to Notion

一个 Chrome 浏览器扩展，一键将 Twitter 推文保存到 Notion 数据库，支持文本、图片、引用推文等内容。

---

## ✅ 功能亮点

- 🔗 一键保存推文到 Notion 数据库
- 🖼️ 支持保存图片、视频、引用推文
- 🏷️ 支持自定义分类（多选）
- 🧵 支持长文本自动分块
- 📊 自动记录点赞、转推、评论数
- 🕒 自动记录发布时间和保存时间

---

## 🧩 安装方式

### 方法 ：开发者模式加载（推荐）

1. 下载本仓库并解压
2. 打开 Chrome → 扩展程序 → 开发者模式
3. 点击「加载已解压的扩展程序」
4. 选择本项目文件夹即可
